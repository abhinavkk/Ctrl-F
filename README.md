## INSTALLATION
##### 1. Install Python3
##### 2. Install Flask-socketio
##### 3. Install Flast-wtf
##### 4. Install selenium

## RUN THE APPLICATION
###### 1. Clone the project
###### 2. Go to the `Ctrl-F` project folder
###### 3. Run the  `app.py` script
###### 4. Open the Chrome browser and go to `http://127.0.0.1:5000`
